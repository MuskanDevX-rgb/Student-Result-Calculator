# 🎓 Student Result Calculator

A modern and responsive Student Result Calculator built using HTML, CSS, and JavaScript.

## 🚀 Features

- Student Name Input
- 5 Subject Marks
- Total Marks Calculation
- Percentage Calculation
- Grade Calculation
- Performance Status
- Pass/Fail Result
- Animated Progress Bar
- Dark Mode
- Live Date & Time
- Print Result
- Responsive UI

## 🛠 Technologies Used

- HTML5
- CSS3
- JavaScript
